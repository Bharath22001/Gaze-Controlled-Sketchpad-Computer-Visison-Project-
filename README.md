Here you will find a computer vision project I embarked on. The main highlight is the gaze controlled sketchpad which as the same suggests, allows the user to draw on the sketchpad using their eyes. Accompanied with facial expression detection to identify when to start/stop drawing, this application calculates the trajectory of the user's gaze and draws where they are looking on the sketchpad.
